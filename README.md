# JSMarks

Ссылка на сайт: https://it.steamify.work

Простое веб-приложение для заметок и списков задач.  
Все данные хранятся в браузере — никаких серверов и регистрации.  
Уникальная фишка: **поделиться заметкой по короткой ссылке** прямо из приложения.

---

## Что умеет приложение

| Возможность     | Описание                                       |
|-----------------|------------------------------------------------|
| **Заметки**     | Текстовые заметки с форматированием            |
| **Списки**      | Чекбоксы, выполнение — одним кликом            |
| **Закрепление** | Важные заметки всегда вверху                   |
| **Теги**        | Группировка и фильтрация по темам              |
| **Цвета**       | 6 цветов карточек — для визуального разделения |
| **Поиск**       | По заголовку, содержимому и тегам              |
| **Шеринг**      | Короткая ссылка на любую заметку               |
| **Темы**        | Тёмная и светлая тема + 3 шрифта               |
| **Бэкап**       | Экспорт и импорт всех данных в JSON            |
| **Кроссбраузерность** | Работает в любых браузерах                     |

## Инструкция для пользователя

### Создание заметки
1. Нажмите **«+»** на вкладке «Заметки»
2. Введите заголовок и текст
3. Добавьте теги через запятую (например: `работа, идеи`)
4. Выберите цвет карточки
5. При желании закрепите заметку
6. Нажмите **«Сохранить»**

### Создание списка
1. Перейдите на вкладку **«Списки»**
2. Нажмите **«+»**
3. Введите заголовок и каждый пункт с новой строки
4. Добавьте теги через запятую (например: `работа, идеи`)
5. Выберите цвет карточки
6. При желании закрепите заметку
7. Сохраните — пункты появятся с чекбоксами

### Управление

| Действие | Как сделать                                             |
|----------|---------------------------------------------------------|
| **Открыть заметку** | Кликните по карточке                                    |
| **Редактировать** | Кликните по карточке и отредактируйте ее                |
| **Удалить** | Кликните по карточке и удалите ее, подтвердите удаление |
| **Закрепить** | Нажмите на иконку 📌 в карточке                         |
| **Поделиться** | Нажмите «Поделиться» и скопируйте ссылку                |

### Поиск и фильтры

| Инструмент                                             | Что делает                                                    |
|--------------------------------------------------------|---------------------------------------------------------------|
| Поле поиска                                            | Ищет по заголовку, тексту, тегам и пунктам задач              |
| Фильтр «Все/Только закреплённые/Только незакрепленные» | Показывает только закреплённые/незакрепленные или все заметки |
| Фильтр по тегу                                         | Введите тег для фильтрации (с автодополнением)                |
| Сортировка                                             | По дате редактирования или алфавиту                           |
| Вид                                                    | Плитка / список (кнопки в виде иконок)                        |
| Сброс фильтров                                         | Сбрасывает все фильтры и поиск разом                          |

### Настройки ⚙️
- **Тёмная / светлая тема**
- **Шрифт** (Comfortaa / Nunito / Roboto)
- **Экспорт** — сохранить все заметки в файл
- **Импорт** — восстановить из файла
- **Удалить всё** — полностью очистить хранилище

### Шеринг заметок (уникальная фича)
1. Нажмите **«Поделиться»** на любой карточке
2. Скопируйте ссылку
3. Отправьте другу
4. При переходе откроется окно просмотра (только чтение)

> Вся информация о заметке кодируется прямо в ссылке. Сервер не нужен.

### Как шеринг устроен (технически)
Ссылка выглядит так:
https://...../index.html#s=eyJ0Ijoi0JrRg9C_0LjRgtGM...
1. Всё, что после #s= — это закодированные данные заметки
2. При открытии ссылки JavaScript читает эту часть и показывает заметку
3. Сервер получает запрос только на страницу, хеш остаётся в браузере
#### Нюансы:
- Длина ссылки — до 2000 символов (хватает для обычных заметок)
- Данные не зашифрованы (только закодированы), не стоит передавать пароли
- Ссылка показывает состояние заметки на момент создания

## Техническая информация (для разработчиков)

### Стек

| Технология | Назначение |
|------------|------------|
| HTML5 | Структура страниц |
| CSS3 | Стили, адаптив, темы, кастомный скроллбар |
| JavaScript (ES6+) | Вся логика приложения |
| localStorage | Хранение данных (до 10 МБ) |

### Структура проекта

jsmarks/
- index.html # Главная страница
- main.css # Все стили (включая темы)
- main.js # Полная логика приложения
- README.md # Документация
- TS.md # Техническое задание на разаработку

### Основные функции в `main.js`

| Функция | Что делает                              |
|---------|-----------------------------------------|
| `renderNotes()` / `renderTasks()` | Отрисовка карточек заметок и задач      |
| `saveToLocal()` / `loadData()` | Сохранение и загрузка из localStorage   |
| `generateLink()` / `showSharedNote()` | Генерация ссылки и показ шеринга        |
| `showNote()` | Открытие заметки с активными чекбоксами |
| `filterBySearch()` | Поиск по тексту и тегам                 |
| `getVisibleItemsForTab()` | Фильтрация + сортировка                 |
| `openModalForEdit()` | Форма редактирования заметки            |
